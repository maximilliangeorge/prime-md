# Not how the world is, is the mystical, but that it is

Not how the world is, is the mystical, but *that* it is. (6.44)

If [ethics is transcendental](./ethics-is-transcendental.md) — if value lies
outside the world of facts — then the deepest question is not any question about
how the world is configured, but the sheer wonder that there is a world at all.
And if [language can only describe how the world is, not that it
is](./limits-of-language.md), then this wonder is inexpressible.

The contemplation of the world *sub specie aeterni* is its contemplation as a
limited whole. The feeling of the world as a limited whole is the mystical
feeling. (6.45)

For an answer which cannot be expressed the question too cannot be
expressed. (6.5) We feel that even if all possible scientific questions be
answered, the problems of life have still not been touched at all. (6.52)

There is indeed the inexpressible. This shows itself; it is the mystical. (6.522)
